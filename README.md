# V2Ray-SiteDAT

用于 Project V 的自定义域名文件。
此项目用于个人列表的编辑与自动生成。
</br>
</br>
## 使用
下载 geofiles 文件夹中的 .dat 域名文件放到 V2Ray 的资源目录中，域名文件的标签及标签对应的域名见同名文件夹下的文件，在V2Ray 配置中按标签设定路由规则。
</br>
</br>
